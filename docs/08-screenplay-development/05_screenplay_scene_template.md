# Screenplay Scene Template

Each screenplay scene must contain:

## Scene Identity

- Scene ID
- Source HWT IDs
- Scene architecture reference
- Location
- Time

## Dramatic Layer

- Character objective
- Conflict
- Emotional turn
- New information revealed

## Writing Layer

- Visual opening
- Action progression
- Dialogue purpose
- Silence moments
- Song/performance function

## Production Layer

- Props
- Costume state
- Camera intention
- Sound design

## Evidence Layer

- LOCKED_SOURCE elements
- LOCKED_CONTINUITY elements
- Development choices
- Caution notes
