# Feature Runtime Architecture

## Principle

Silappathikaram should not be compressed into a conventional hero journey. Runtime follows the epic movement of truth:

**abundance → separation → journey → failed justice → transformation → memory**

## Target format

Feature epic:

- Approximate runtime: 170–190 minutes
- Interval structure: optional after Madurai court collapse
- Scene pacing determined by dramatic function, not equal duration

## Five movements

### Movement I — Pukār

Purpose:
Establish a world worth losing.

### Movement II — Road

Purpose:
Turn choices into physical consequences.

### Movement III — Madurai

Purpose:
Expose institutional failure through evidence.

### Movement IV — Fire

Purpose:
Transform personal grief into moral consequence.

### Movement V — Vañci

Purpose:
Show how societies preserve memory.
