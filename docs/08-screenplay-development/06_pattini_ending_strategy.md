# Pattini Ending Strategy

## Core principle

The ending must transform an individual tragedy into collective memory without erasing the original injustice.

## Final movement

### Witness

Hill communities recognise Kaṇṇaki before royal institutions act.

### Transmission

Sāttanār, Māṭalan and other witnesses carry the event across social worlds.

### Institution

Senguttuvan creates a memorial structure around an already established moral truth.

### Final image principle

The conclusion should hold together:

- grief;
- justice;
- remembrance;
- ethical instruction.

Avoid ending as only triumph, revenge or divine spectacle.
