# Material Culture Bible — Index

## Purpose

This index governs recurring objects across art direction, props, costume, performance, sound, VFX and continuity.

Objects carry wealth, labour, evidence, memory, authority and ritual action. Each part uses the status labels `LOCKED_SOURCE`, `HISTORICAL_RESEARCH`, `DEVELOPMENT` and `CAUTION`.

## Governing rules

1. Every hero object requires source IDs and scene IDs.
2. Historical form and creative interpretation must remain separately labelled.
3. Repeated objects require ownership and condition history.
4. Ritual objects from different communities must not be merged into a generic sacred package.
5. No hero object is production-locked without performer, safety, continuity, sound and cultural review.
