A chapter-by-chapter English translation of Dr. Sundara Shanmuganar’s Tamil literary study of *Cilappatikāram*.

## Reading order

- `00-front-matter-and-entrance.md`
- `01-epic-of-the-three-crowned-kings.md` through `31-solutions-to-several-problems.md`

A single combined file is available one level above as [`../silambo-silambu-complete-supplied-text.md`](../silambo-silambu-complete-supplied-text.md).
